# Flow Bot Builder

SPA React + Tailwind CSS để phác thảo flow bot bằng kéo thả.

## Chạy local

```bash
npm install
npm run dev
```

Mở `http://127.0.0.1:5173`.

## Tính năng hiện có

- 5 node cơ bản: Start, Message, Question, Condition, Action.
- Kéo node từ palette vào canvas.
- Kéo node để sắp xếp lại flow.
- Bấm output port rồi input port để nối node.
- Chỉnh label/data trong Inspector.
- Build, copy hoặc tải JSON mô tả kịch bản bot.
